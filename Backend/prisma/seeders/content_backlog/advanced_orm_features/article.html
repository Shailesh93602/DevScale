Advanced ORM and ODM Features

Beyond simple CRUD and associations, modern ORMs like Sequelize and ODMs like Mongoose offer advanced techniques for performance, security, and scalability.

1. Raw Queries

Sometimes the ORM's abstraction gets in the way of highly optimized SQL. Both Sequelize and Mongoose allow you to run raw queries if necessary.


// Sequelize Raw Query
const [results, metadata] = await sequelize.query("SELECT * FROM users WHERE active = true");

2. Scopes in Sequelize

Scopes are predefined sets of criteria that can be easily applied to models for cleaner and more reusable code.


const User = sequelize.define('User', { ... }, {
  scopes: {
    active: { where: { status: 'active' } },
    over18: { where: { age: { [Op.gte]: 18 } } }
  }
});

// Use it like this:
const activeUsers = await User.scope('active').findAll();

3. Static and Instance Methods in Mongoose

Mongoose allows you to define custom logic directly on your schema.


userSchema.statics.findByEmail = function(email) {
  return this.find({ email: email });
};

4. Hooks and Middlewares

Lifecycle hooks (Sequelize) and middleware (Mongoose) allow you to inject logic at specific points in the data lifecycle, such as password hashing before Save or logging after Delete.

Conclusion

Mastering these advanced features allows you to push the boundaries of what your ORM can do, resulting in cleaner, more maintainable, and higher-performing applications.
