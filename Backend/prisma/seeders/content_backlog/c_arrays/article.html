Understanding Arrays in C

An array is a data structure that can store a fixed-size sequential collection of elements of the same type. It is used to represent a collection of data of a certain type.

1. Declaration and Initialization

To declare an array, you specify the data type and the number of elements: int scores[5];. You can also initialize it at declaration: int scores[5] = {90, 85, 70, 95, 80};

2. Zero-Based Indexing

Crucially, C uses zero-based indexing. The first element is at index 0, the second at index 1, and the last element (in an array of size N) is at index N-1.

3. Accessing and Modifying

You can access or modify an element using its index: int firstScore = scores[0]; or scores[1] = 88;. Be careful not to access an index outside the array bounds, as C does not perform 'bounds checking'.

Conclusion

Arrays are one of the most important concepts in programming, providing a way to organize and manipulate sets of related data efficiently.
