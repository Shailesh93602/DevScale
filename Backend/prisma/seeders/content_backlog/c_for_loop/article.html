Understanding the For Loop in C

A for loop is a repetition control structure that allows you to efficiently write a loop that needs to execute a specific number of times.

1. The Three Parts of a For Loop

  • Initialization: Executed only once at the start (e.g., int i = 0;).
  • Condition: Evaluated before every iteration. If false, the loop ends (e.g., i < 10;).
  • Increment/Decrement: Executed after every iteration (e.g., i++).

2. Syntax

for (initialization; condition; increment) {
    // block of code to be executed
}

3. When to use a For Loop?

The for loop is ideal when you know exactly how many times you want to iterate (e.g., through an array of known size). If you don't know the number of iterations in advance, a while loop might be more appropriate.

Conclusion

The for loop is the workhorse of iterative programming, providing a clean and organized way to repeat tasks in your code.
