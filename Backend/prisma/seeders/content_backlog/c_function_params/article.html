Understanding Function Parameters in C

Parameters act as placeholders for the data (arguments) that you pass into a function, allowing it to perform tasks using different values.

1. Passed by Value

By default, C passes parameters **by value**. This means the function receives a copy of the data. Changes made to the parameter inside the function do NOT affect the original variable outside.

2. Passing Multiple Parameters

Functions can accept any number of parameters, separated by commas. Each parameter must have its own data type and name: void greet(char name[], int age);.

3. Returning Values

A function can send data back to the caller using the return statement. The data type of the returned value must match the return type specified in the function's declaration.

Conclusion

Parameters and return values are the communication channels between your functions, allowing them to collaborate and share information to achieve complex goals.
