Understanding the If Statement in C

The if statement is the most basic control structure, allowing your program to execute a block of code ONLY if a specific condition is true.

1. Truth and Falsehood

In C, any value that is **Non-Zero** is considered true, and a value of **Zero** is considered false. The condition inside the if statement is always evaluated in this way.

2. Basic Syntax

if (condition) {
    // block of code to be executed if the condition is true
}

3. Comparison Operators

Conditions are typically built using comparison operators like == (equal to), != (not equal to), >, <, etc. For example: if (age >= 18).

Conclusion

The if statement is the fundamental building block of logic in C, enabling your programs to make decisions and react to different data values.
