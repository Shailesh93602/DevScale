Understanding Output and New Lines in C

The printf() function is the most common way to display output on the screen in C. To organize this output, we use special characters for new lines.

1. The printf() Function

printf() stands for "print formatted". It takes a string as its first argument and any number of variables to fill in the format specifiers. For example: printf("Hello, World!");

2. The Newline Character (\n)

By default, printf() does not add a new line at the end of its output. To move the cursor to the next line, you must explicitly include the \n escape sequence. For example: printf("Line 1\nLine 2");

3. Multiple Newlines

You can use multiple \n characters to create blank lines: printf("Hello\n\nWorld"); will leave a one-line gap between the words.

Conclusion

Using printf() and \n correctly is the first step toward creating readable and professional-looking command-line applications in C.
