Reading from Files in C

To read data from a file, you must open it in read ("r") mode and use functions that pull data from the file stream into your variables.

1. Using fgets()

The most common way to read a file line by line. It reads characters from the file and stores them into a string until it reaches a newline or a specific length: fgets(buffer, 100, fptr);.

2. Using fscanf()

Works like scanf() but for files. Ideal for reading structured data (like numbers separated by spaces). For example: fscanf(fptr, "%d %f", &age, &price);.

3. The EOF (End Of File) Marker

Most reading functions return a special value, EOF, when they reach the literal end of the file. You can use this in a loop to read an entire file, regardless of its size.

Conclusion

Reading from files is what enables your applications to load configurations, process large data sets, and resume previous work by loading saved state.
