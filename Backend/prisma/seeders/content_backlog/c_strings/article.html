Understanding Strings in C

A string in C is not a separate data type but is represented as a character array that is terminated by a special Null Character (\0).

1. Strings are Character Arrays

You declare a string just like any other array: char name[] = "EduScale";. The compiler automatically adds the \0 at the end, so a string with 8 characters actually takes up 9 bytes of memory.

2. Input and Output

You can print a string using the %s format specifier: printf("%s", name);. To read a string from the user, you use scanf("%s", name); (note: scanf stops at the first space it encounters).

3. Immutability?

Unlike languages where strings are immutable, in C, you can modify any individual character in a string array directly (e.g., name[0] = 'e';).

Conclusion

Because strings are just arrays, all your knowledge of indices and loops applies directly to them. This provides deep control over text processing, but requires care to manage memory correctly.
