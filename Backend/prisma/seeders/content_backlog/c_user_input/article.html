Handling User Input in C

Most programs need to interact with the user to get data. In C, the primary way to do this at the command line is the scanf() function.

1. Using scanf()

The scanf() function reads formatted input from the standard input (keyboard). You must provide a Format Specifier and the Address of the variable where the data should be stored.

int age;
scanf("%d", &age); // Note the '&' symbol!

2. The Address Operator (&)

The & symbol is the address-of operator. It tells scanf exactly where in the computer's memory to store the value it reads. Forgetting this symbol is a common beginner mistake that leads to crashes.

3. Multiple Inputs

You can read multiple values in a single scanf call by using multiple format specifiers: scanf("%d %f", &myInt, &myFloat);.

Conclusion

Mastering user input allows you to build dynamic, interactive programs that can respond to real-world data and user choices.
