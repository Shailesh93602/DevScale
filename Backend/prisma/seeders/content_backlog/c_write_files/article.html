Writing to Files in C

Once you have a file open in write ("w") or append ("a") mode, you can use several functions to send data into it.

1. Using fprintf()

This is the most flexible function. It works exactly like printf(), but its first argument is the file pointer: fprintf(fptr, "Hello, %s!", name);.

2. Using fputs()

Used specifically for writing strings to a file. It is simple and efficient: fputs("This is a line of text.", fptr);.

3. Flushing the Buffer

When you write to a file, C often stores the data in a temporary 'buffer' in memory first. To ensure all data is actually written to the physical disk, use fflush(fptr); or simply fclose(fptr);.

Conclusion

Writing data to files is a fundamental skill for creating durable and useful software that can produce output beyond the terminal screen.
