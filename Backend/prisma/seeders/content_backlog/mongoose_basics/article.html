Introduction to Mongoose and MongoDB

Mongoose is an Object Data Modeling (ODM) library for MongoDB and Node.js. It provides a straight-forward, schema-based solution to model your application data.

1. Installation

To use Mongoose, you simply need to install the package.


npm install mongoose

2. Connecting to MongoDB


const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/mydatabase', {
  useNewUrlParser: true,
  useUnifiedTopology: true
});

3. Defining Schemas and Models

Schemas define the structure of the documents within a collection, while models are a constructor compiled from a Schema.


const { Schema } = mongoose;

const userSchema = new Schema({
  name: String,
  email: { type: String, required: true },
  age: Number
});

const User = mongoose.model('User', userSchema);

4. Creating and Querying Documents


// Create
const user = new User({ name: 'John Doe', email: 'john@example.com' });
await user.save();

// Read
const users = await User.find({ age: { $gt: 18 } });

Conclusion

Mongoose brings structure and validation to MongoDB's schema-less nature. Once connected and with models defined, you can easily perform CRUD operations and manage your data with ease.
