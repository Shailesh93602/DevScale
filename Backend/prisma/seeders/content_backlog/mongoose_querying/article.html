Advanced Mongoose Querying

Mongoose provides a rich API for querying MongoDB, from simple find operations to complex aggregations.

1. Basic and Advanced Queries


// Find by ID
const user = await User.findById('60c...');

// Find with complex criteria using operators
const users = await User.find({
  age: { $gte: 18, $lte: 65 },
  status: 'active'
});

2. Sorting and Pagination

Sorting and pagination are straightforward in Mongoose with sort, limit, and skip.


const users = await User.find()
  .sort({ createdAt: -1 })
  .skip(10)
  .limit(20);

3. Population

Population is the process of automatically replacing the specified paths in the document with document(s) from other collection(s).


const post = await Post.findOne().populate('author');
console.log(post.author.name);

4. The Query Builder

Mongoose methods often return a Query object, which allows you to chain methods like select, where, and sort before execution.

Conclusion

Flexible querying and population are what make Mongoose so powerful for building real-world applications. Master these techniques to build advanced data-driven features efficiently.
