List Comprehensions and Generators

Python provides several ways to create and transform sequences. While list comprehensions are fast, **Generators** are the ultimate tool for memory efficiency.

1. List Comprehension Recap

A concise way to create lists: [x**2 for x in my_list]. It builds the ENTIRE list in memory at once.

2. Introducing Generators

A Generator expression looks like a list comprehension but uses round brackets: (x**2 for x in my_list). Instead of creating a full list, it returns an **iterator** that generates one item at a time only when you ask for it.

3. The 'yield' Keyword

You can also create generators using functions and the yield keyword. When a function yields a value, its state is "paused", allowing it to resume exactly where it left off next time it's called.

Conclusion

Use list comprehensions for small datasets where speed is priority. Use generators for massive or infinite datasets to keep your memory usage near zero.
