Advanced Querying with Sequelize

Sequelize provides a powerful API for querying your database, including advanced operations like filtering, sorting, and pagination.

1. Simple Queries


// Find all users
const users = await User.findAll();

// Find one user by primary key
const user = await User.findByPk(1);

// Find one user matching criteria
const user = await User.findOne({ where: { email: 'john@example.com' } });

2. Filtering with Operators

Sequelize uses the Op object for advanced filtering, which prevents SQL injection and provides a more readable syntax.


const { Op } = require('sequelize');

const users = await User.findAll({
  where: {
    age: {
      [Op.gt]: 18 // age > 18
    }
  }
});

3. Sorting and Pagination

Sorting is handled with the order attribute, and pagination uses limit and offset.


const users = await User.findAll({
  order: [['createdAt', 'DESC']],
  limit: 10,
  offset: 20
});

4. Selecting Specific Attributes

You can choose which columns to retrieve by specifying the attributes property.


const users = await User.findAll({
  attributes: ['firstName', 'email']
});

Conclusion

By leveraging Sequelize's querying API, you can write expressive and performant database interactions without writing a single line of raw SQL. Master filtering, sorting, and pagination to take full control of your application's data layer.
