Testing Asynchronous Code with Mocha

Using the powerful and descriptive asynchronous testing capabilities provided by Mocha, you can create a suite of unit tests for your project. By following these steps, you can set it up in minutes.

1. Testing via Callback


it('should complete asynchronously', function(done) {
  setTimeout(() => {
    done();
  }, 100);
});

2. Testing via Promise


it('should return a promise', function() {
  return someAsyncFunction().then(res => {
    expect(res).to.equal(expected);
  });
});

3. Testing via async/await


it('should work with async/await', async function() {
  const res = await someAsyncFunction();
  expect(res).to.equal(expected);
});

Conclusion

By making unit testing a priority, you can ensure that your application's most valuable asset—its code—remains safe and performant for your users.
