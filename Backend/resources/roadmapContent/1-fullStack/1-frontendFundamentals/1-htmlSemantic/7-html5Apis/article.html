HTML5 APIs & Browser Features

HTML5 introduced dozens of powerful browser APIs that allow web applications to do things previously only possible in native apps — without any plugins. These APIs are accessed through JavaScript but are part of the browser environment.

Web Storage API

Two storage mechanisms for persisting data client-side:

// localStorage — persists indefinitely (survives browser restarts)
localStorage.setItem('theme', 'dark');
const theme = localStorage.getItem('theme');   // 'dark'
localStorage.removeItem('theme');
localStorage.clear();  // remove all items

// sessionStorage — cleared when the tab is closed
sessionStorage.setItem('cart', JSON.stringify({ items: 3 }));
const cart = JSON.parse(sessionStorage.getItem('cart'));

Both store strings only. Use JSON.stringify() and JSON.parse() for objects. Max ~5–10MB per origin.

Geolocation API

if ('geolocation' in navigator) {
  navigator.geolocation.getCurrentPosition(
    (position) => {
      const { latitude, longitude, accuracy } = position.coords;
      console.log(`You are at ${latitude}, ${longitude}`);
    },
    (error) => {
      console.error('Location error:', error.message);
    },
    { enableHighAccuracy: true, timeout: 5000, maximumAge: 0 }
  );
}

// Watch continuous position updates (great for maps)
const watchId = navigator.geolocation.watchPosition(successCallback);
navigator.geolocation.clearWatch(watchId); // stop watching

History API

Manipulate browser history without full page refreshes — the foundation of Single Page Applications (SPAs):

// Push a new URL to history (no page reload)
history.pushState({ page: 'about' }, 'About', '/about');

// Replace current history entry
history.replaceState({ page: 'home' }, 'Home', '/');

// Navigate programmatically
history.back();    // same as browser back button
history.forward();
history.go(-2);    // go 2 steps back

// Listen for back/forward navigation
window.addEventListener('popstate', (event) => {
  console.log('Navigated to:', event.state);
});

Canvas API

The <canvas> element provides a 2D drawing surface for graphics, charts, games, and image manipulation:

<canvas id="myCanvas" width="400" height="200"></canvas>

<script>
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

// Draw a gradient rectangle
const gradient = ctx.createLinearGradient(0, 0, 400, 0);
gradient.addColorStop(0, '#6366f1');
gradient.addColorStop(1, '#8b5cf6');
ctx.fillStyle = gradient;
ctx.fillRect(0, 0, 400, 200);

// Draw text
ctx.fillStyle = 'white';
ctx.font = 'bold 32px Inter, sans-serif';
ctx.textAlign = 'center';
ctx.fillText('Hello Canvas!', 200, 110);
</script>

Web Workers

Web Workers run JavaScript in a background thread — keeping the main UI thread unblocked during heavy computation:

// main.js
const worker = new Worker('worker.js');
worker.postMessage({ numbers: [1, 2, 3, 4, 5, 1000000] });
worker.onmessage = (event) => {
  console.log('Sum from worker:', event.data.result);
};

// worker.js
self.onmessage = function(event) {
  const sum = event.data.numbers.reduce((a, b) => a + b, 0);
  self.postMessage({ result: sum });
};

Drag and Drop API

<div draggable="true" id="draggable">Drag me!</div>
<div id="dropzone">Drop here</div>

<script>
const draggable = document.getElementById('draggable');
const dropzone = document.getElementById('dropzone');

draggable.addEventListener('dragstart', (e) => {
  e.dataTransfer.setData('text/plain', 'dragged-item');
});

dropzone.addEventListener('dragover', (e) => e.preventDefault()); // allow drop
dropzone.addEventListener('drop', (e) => {
  e.preventDefault();
  const data = e.dataTransfer.getData('text/plain');
  dropzone.textContent = `Received: ${data}`;
});
</script>

Intersection Observer API

Efficiently detect when elements enter/leave the viewport — perfect for lazy loading and infinite scroll:

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
      observer.unobserve(entry.target); // stop observing after first trigger
    }
  });
}, { threshold: 0.1 }); // trigger when 10% visible

document.querySelectorAll('.animate-on-scroll').forEach(el => {
  observer.observe(el);
});

Summary of Key HTML5 APIs

  • localStorage / sessionStorage — Client-side data persistence
  • Geolocation — Device location access
  • History API — SPA routing without page reloads
  • Canvas — 2D graphics and image manipulation
  • Web Workers — Background JavaScript threads
  • Drag & Drop — Native drag interactions
  • Intersection Observer — Viewport visibility detection
  • Fetch API — Modern HTTP requests (replaces XMLHttpRequest)
