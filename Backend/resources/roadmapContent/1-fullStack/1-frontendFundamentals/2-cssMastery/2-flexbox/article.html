CSS Flexbox — The Complete Guide

Flexbox is a one-dimensional layout method designed for placing items in rows or columns. It solves common layout problems that were painful with floats and tables.

Enabling Flexbox

.container {
  display: flex;
}

All direct children become flex items. The container is the flex container.

Main Axis vs Cross Axis

Flexbox has two axes. justify-content aligns along the main axis; align-items aligns along the cross axis.

.container {
  flex-direction: row;    /* Main axis: horizontal (default) */
  flex-direction: column; /* Main axis: vertical */
}

Container Properties

.container {
  display: flex;
  flex-direction: row | column | row-reverse | column-reverse;
  flex-wrap: nowrap | wrap | wrap-reverse;
  justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
  align-items: stretch | flex-start | flex-end | center | baseline;
  align-content: flex-start | center | space-between; /* Multi-line only */
  gap: 16px;                  /* Space between items */
  gap: 16px 8px;              /* Row gap, column gap */
}

Item Properties

.item {
  flex-grow: 1;       /* Grow to fill available space */
  flex-shrink: 1;     /* Shrink if needed */
  flex-basis: auto;   /* Initial size before flex adjustments */
  flex: 1;            /* Shorthand: grow:1, shrink:1, basis:0 */
  flex: 0 0 200px;    /* Fixed 200px, no grow/shrink */

  align-self: center; /* Override align-items for this one item */
  order: 2;           /* Reorder visually without changing HTML */
}

Common Patterns

/* Perfect centering */
.center { display: flex; justify-content: center; align-items: center; }

/* Navbar: logo left, links right */
.navbar { display: flex; justify-content: space-between; align-items: center; }

/* Equal-width columns */
.columns { display: flex; }
.column  { flex: 1; }

/* Responsive wrapping card grid */
.grid { display: flex; flex-wrap: wrap; gap: 16px; }
.card { flex: 1 1 280px; } /* Each card is min 280px, grows equally */
