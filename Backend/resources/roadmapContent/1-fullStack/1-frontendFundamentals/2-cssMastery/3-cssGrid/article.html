CSS Grid — Two-Dimensional Layouts

CSS Grid is a two-dimensional layout system handling both rows and columns simultaneously — perfect for page-level layouts. Use Grid for the big picture; use Flexbox for aligning within components.

Defining a Grid

.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px; /* 3 columns */
  grid-template-rows: auto 1fr auto;      /* 3 rows */
  gap: 20px;
}

The fr Unit

The fr (fraction) unit distributes available space. 1fr 2fr means one column gets 1/3 and the other gets 2/3 of available space.

repeat() & minmax()

/* 12 equal columns */
grid-template-columns: repeat(12, 1fr);

/* Auto-responsive grid — no media queries needed! */
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

Placing Items Explicitly

.header {
  grid-column: 1 / -1;  /* Span full width (1 to last line) */
}
.sidebar {
  grid-column: 1 / 2;
  grid-row: 2 / 3;
}
.main {
  grid-column: 2 / 4;   /* Span columns 2 to 4 */
  grid-row: 2 / 3;
}
/* Shorthand */
.item { grid-area: 2 / 1 / 3 / 3; } /* row-start/col-start/row-end/col-end */

Named Grid Areas

.layout {
  display: grid;
  grid-template-columns: 240px 1fr;
  grid-template-rows: 60px 1fr 60px;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
}
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

Alignment in Grid

.container {
  justify-items: start | end | center | stretch;  /* Align items in column */
  align-items:   start | end | center | stretch;  /* Align items in row */
  justify-content: start | center | space-between; /* Align grid in container */
  align-content:   start | center | space-between;
}
.item {
  justify-self: center; /* Override for single item */
  align-self: end;
}

Grid vs Flexbox — When to Use Each

  • Use Grid for 2D page-level layouts (header, sidebar, main, footer).
  • Use Flexbox for 1D component-level alignment (nav links, card contents).
  • They are complementary — use both on the same page!
