JavaScript Variables, Data Types & Operators

JavaScript is a dynamically typed language — variables can hold any type of value, and types are determined at runtime, not at declaration time.

Declaring Variables

// const — block-scoped, cannot be reassigned (preferred default)
const PI = 3.14159;
PI = 3; // ❌ TypeError: Assignment to constant variable

// let — block-scoped, can be reassigned
let score = 0;
score = 10; // ✅

// var — function-scoped, hoisted (avoid in modern JS)
var legacyVar = "old style"; // has confusing hoisting quirks

Primitive Data Types (7 total)

let str    = "Hello World";       // String
let num    = 42;                  // Number (includes floats)
let big    = 9007199254740991n;   // BigInt
let bool   = true;               // Boolean
let nil    = null;               // Null — intentional empty value
let undef  = undefined;          // Undefined — not yet assigned
let sym    = Symbol("uniqueId"); // Symbol — always unique

Reference Types

// Objects, Arrays, Functions are reference types
const user = { name: "Alice", age: 30 };
const nums = [1, 2, 3, 4, 5];
const greet = () => "Hello!";

Type Checking

typeof "hello"      // "string"
typeof 42           // "number"
typeof true         // "boolean"
typeof undefined    // "undefined"
typeof null         // "object"  ← historic bug in JS!
typeof []           // "object"
typeof {}           // "object"
typeof function(){} // "function"

// Better checks:
Array.isArray([])         // true
value === null            // for null
value instanceof Date     // for instances

= vs == vs ===

=   // Assignment
==  // Loose equality (performs type coercion — avoid!)
=== // Strict equality (checks both value AND type — use this!)

0 == "0"    // true  (coercion!)
0 === "0"   // false (different types)
null == undefined   // true  (special case)
null === undefined  // false

Modern Operators

// Optional chaining — safe nested access
const city = user?.address?.city; // undefined instead of crash

// Nullish coalescing — default only if null/undefined
const name = user.name ?? "Guest"; // better than || for 0/""/false

// Logical assignment
user.role ??= "user";    // assign if null/undefined
counter &&= counter + 1; // assign if truthy
value ||= "default";     // assign if falsy
