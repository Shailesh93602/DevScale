JavaScript Operators

Operators are special symbols used to perform operations on variables and values. JavaScript includes several types of operators, from basic arithmetic to complex bitwise operations.

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations:

  • + (Addition): Adds two numbers. Also used for string concatenation.
  • - (Subtraction): Subtracts one number from another.
  • * (Multiplication): Multiplies two numbers.
  • / (Division): Divides one number by another.
  • % (Modulus): Returns the division remainder.
  • ** (Exponentiation): Raises the first operand to the power of the second (ES2016).
  • ++ (Increment): Increases a value by 1.
  • -- (Decrement): Decreases a value by 1.

2. Assignment Operators

Assignment operators assign values to JavaScript variables:

  • = (Assignment): x = y
  • += (Add and Assign): x += y (same as x = x + y)
  • -= (Subtract and Assign): x -= y (same as x = x - y)
  • *= (Multiply and Assign): x *= y
  • /= (Divide and Assign): x /= y

3. Comparison Operators

Comparison operators are used in logical statements to determine equality or difference between variables or values. They return a boolean (true or false).

  • ==: Equal to (performs type coercion).
  • ===: Strict equal to (checks both value and type).
  • !=: Not equal to.
  • !==: Strict not equal to.
  • >, <: Greater than, Less than.
  • >=, <=: Greater than or equal to, Less than or equal to.

4. Logical Operators

Logical operators are used to determine the logic between variables or values:

  • && (Logical AND): Returns true if both operands are true.
  • || (Logical OR): Returns true if at least one operand is true.
  • ! (Logical NOT): Reverses the boolean value of its operand.

5. Conditional (Ternary) Operator

The only operator that takes three operands. It is frequently used as a shortcut for the if statement:

let access = (age > 18) ? "Allowed" : "Denied";

6. Type Operators

  • typeof: Returns the type of a variable.
  • instanceof: Returns true if an object is an instance of an object type.

Conclusion

Mastering operators is crucial for controlling logic and data flow in your applications. Always prefer strict comparison (===) to avoid unexpected type coercion bugs.
