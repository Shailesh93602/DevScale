JavaScript Control Structures

Control structures allow you to control the flow of execution in your code based on conditions. The most common control structures in JavaScript are if...else and switch statements.

1. If...Else Statements

The if statement executes a block of code if a specified condition is true. The else statement executes if the condition is false.

let hour = 14;

if (hour < 12) {
  console.log("Good morning");
} else if (hour < 18) {
  console.log("Good afternoon");
} else {
  console.log("Good evening");
}

2. Switch Statement

The switch statement is used to perform different actions based on different conditions. It is a more efficient alternative to multiple if...else if statements when comparing the same expression against many values.

let day;
switch (new Date().getDay()) {
  case 0:
    day = "Sunday";
    break;
  case 1:
    day = "Monday";
    break;
  case 6:
    day = "Saturday";
    break;
  default:
    day = "Weekday";
}

The Importance of `break`

Without the break keyword, the script will continue to execute the next case even if the evaluation does not match the case. This is called "falling through."

3. Truthy and Falsy Values

In JavaScript, every value has an inherent boolean value when used in a conditional context:

  • Falsy values: false, 0, "" (empty string), null, undefined, and NaN.
  • Truthy values: Everything else, including "0", "false", empty arrays [], and empty objects {}.

Conclusion

Control structures are the logical heart of your programs. By combining these with comparison and logical operators, you can build complex decision-making processes in your web applications.
