JavaScript Loop Control

Loops are used to execute a block of code multiple times. In JavaScript, there are several types of loops to handle different iteration needs.

1. The For Loop

The for loop is ideal when you know exactly how many times you want to run a block of code.

for (let i = 0; i < 5; i++) {
  console.log("Iteration number: " + i);
}

2. The While Loop

The while loop executes a block of code as long as a specified condition remains true.

let count = 0;
while (count < 3) {
  console.log("Count is: " + count);
  count++;
}

3. The Do...While Loop

The do...while loop is similar to the while loop, but it guarantees the code block will execute at least once before the condition is checked.

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 0); // Executes once even though condition is false

4. Loop Control: Break and Continue

  • break: Used to exit a loop prematurely when a certain condition is met.
  • continue: Used to skip the rest of the current iteration and move to the next cycle of the loop.
for (let i = 0; i < 10; i++) {
  if (i === 3) continue; // Skip 3
  if (i === 6) break;    // Exit loop at 6
  console.log(i); // Outputs 0, 1, 2, 4, 5
}

5. For...in and For...of

Modern JavaScript provides specific loops for iterating over object properties (for...in) and iterable values like arrays (for...of).

const colors = ["red", "green", "blue"];
for (const color of colors) {
  console.log(color);
}

Conclusion

Loops are fundamental for processing collections of data and automating repetitive tasks. Choose the loop that best fits your logic and always ensure your loops have a valid exit condition to avoid infinite loops!
