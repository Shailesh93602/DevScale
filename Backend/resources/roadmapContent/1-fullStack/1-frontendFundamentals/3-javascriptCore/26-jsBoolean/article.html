JavaScript Boolean

A Boolean represents one of two values: true or false. Booleans are frequently used in conditional testing, to determine which block of code should be executed.

1. The Boolean() Function

The Boolean() function can be used to find out if an expression (or a variable) is true. It returns true for all truthy values and false for all falsy values.

Boolean(10 > 9); // Returns true

2. Truthy and Falsy Values

Everything with a real value is true. Everything with no value is false.

  • Falsy values: false, 0, "" (empty string), null, undefined, and NaN.
  • Truthy values: "0", "false", [] (empty array), and {} (empty object).
// Example of truthy check
  if ([]) {
    console.log("Empty array is TRUTHY!");
  }

3. Booleans Can Be Objects (Wait, Don't Use This!)

Normally, booleans are primitive values. But they can also be defined as objects with the new Boolean() constructor. However, this is strongly discouraged because it behaves inconsistently with operators and comparison.

let x = false;          // boolean primitive
  let y = new Boolean(false); // boolean object

  console.log(typeof x); // "boolean"
  console.log(typeof y); // "object"
  console.log(x == y);   // true
  console.log(x === y);  // false (beware!)

Conclusion

Boolean values are the simplest but most important data type in programming logic. Understanding what values in JavaScript evaluate to true and false is essential for writing error-free conditional logic.
