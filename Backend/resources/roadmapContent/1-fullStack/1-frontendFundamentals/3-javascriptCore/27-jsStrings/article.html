JavaScript Strings

A JavaScript string is zero or more characters written inside quotes. Strings are used for storing and manipulating text.

1. String Literals and Quotes

Strings can be defined using single quotes ('), double quotes ("), or backticks (`).

let s1 = 'Single quote';
  let s2 = "Double quote";
  let s3 = `Template literal (ES6)`;

2. Template Literals (ES6)

Template literals allow us to embed expressions inside strings using ${expression} syntax. This is much cleaner than traditional concatenation.

let name = "Alice";
  console.log(`Hello, my name is ${name}!`);

3. String Properties and Methods

  • length: Returns the number of characters in the string.
  • slice(start, end): Extracts a part of a string and returns it as a new string.
  • substring(start, end): Similar to slice, but doesn't handle negative indexes.
  • replace(target, replacement): Replaces a specific value with another value.
  • toUpperCase() / toLowerCase(): Changes the case of the string.
  • trim(): Removes whitespace from both sides of the string.
  • split(separator): Converts a string into an array based on the separator.

4. Character Access

You can access individuals characters in a string using bracket notation or the charAt() method:

let s = "Hello";
  console.log(s[0]);       // "H"
  console.log(s.charAt(1)); // "e"

Conclusion

Strings are immutable in JavaScript, meaning that string methods always return a new string and do not change the original one. Mastering string manipulation is a daily requirement for any frontend developer.
