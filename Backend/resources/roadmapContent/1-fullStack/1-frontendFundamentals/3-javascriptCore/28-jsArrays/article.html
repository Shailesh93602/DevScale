JavaScript Arrays

Arrays are special objects that allow you to store multiple values in a single variable. They are the most common way to handle ordered collections of data in JavaScript.

1. Creating an Array

The easiest way to create an array is by using the array literal syntax:

const fruits = ["Apple", "Banana", "Cherry"];

2. Accessing Elements

Arrays are zero-indexed, meaning the first element is at index 0, the second at 1, etc.

console.log(fruits[0]); // "Apple"
  fruits[1] = "Blueberry"; // Modifying elements

3. Core Array Methods (Modification)

  • push(val): Adds a value to the end of the array.
  • pop(): Removes the last element.
  • unshift(val): Adds a value to the beginning.
  • shift(): Removes the first element.
  • splice(index, howMany, [items...]): Adds or removes elements from any position.

4. Core Array Methods (Iteration)

Modern JavaScript uses higher-order functions for cleaner code:

  • map(callback): Creates a new array by applying a function to each element.
  • filter(callback): Creates a new array with all elements that pass a test.
  • forEach(callback): Executes a function for each element (returns undefined).
  • find(callback): Returns the first element that passes a test.
const numbers = [1, 2, 3];
  const doubled = numbers.map(n => n * 2); // [2, 4, 6]

Conclusion

Mastering arrays and their iteration methods is essential for almost every modern frontend role. Whether you're handling API data or managing UI state, you'll reach for arrays every single day.
