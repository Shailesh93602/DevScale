JavaScript Date

The Date object is used to work with dates and times in JavaScript. It allows you to create, store, and manipulate timestamps effortlessly.

1. Creating a Date Object

There are four ways to create a new Date object:

  • new Date(): Current date and time.
  • new Date(year, month, day, ...): Specified date.
  • new Date(milliseconds): Milliseconds since Jan 1, 1970.
  • new Date("date string"): Parsed string.
let d = new Date();
  console.log(d.toString()); // Mon Mar 30 2026...

2. Date Methods (Getting)

  • getFullYear(): Returns 4-digit year.
  • getMonth(): Returns month (0-11, where 0 is January).
  • getDate(): Returns day as a number (1-31).
  • getDay(): Returns weekday as a number (0-6).
  • getTime(): Returns milliseconds since Jan 1, 1970.

3. Date Methods (Setting)

Setting methods allow you to modify a Date object:

let d = new Date();
  d.setFullYear(2022); // Change year to 2022

4. Formatting Dates

Modern JavaScript provides the Intl.DateTimeFormat API for high-level formatting, while basic methods like toISOString() are essential for data storage.

let d = new Date();
  console.log(d.toISOString()); // "2026-03-30T09:45:11.412Z"

Conclusion

Always remember that months in the JavaScript Date object are zero-indexed (0 = January). This is one of the most common pitfalls for beginners and even experienced developers working with dates.
