DOM Manipulation with JavaScript

The DOM (Document Object Model) is the browser's in-memory tree representation of your HTML. JavaScript can read and mutate this tree to dynamically update what the user sees without reloading the page.

Selecting Elements

// By ID — fastest, returns one element or null
const hero = document.getElementById("hero");

// By CSS selector — returns first match
const btn = document.querySelector(".btn-primary");
const firstCard = document.querySelector("section > .card");

// By CSS selector — returns all matches as NodeList
const allCards = document.querySelectorAll(".card");
allCards.forEach(card => card.classList.add("loaded"));

Reading & Modifying Content

const heading = document.querySelector("h1");

// Read content
heading.textContent;     // Plain text (strips HTML tags)
heading.innerHTML;       // HTML string including tags

// Write content
heading.textContent = "New Title";           // Safer — no XSS risk
heading.innerHTML = "<span>New</span> Title"; // Parses HTML — ⚠️ XSS risk with user data

Classes & Inline Styles

const el = document.querySelector(".card");

// Class manipulation (preferred over inline styles)
el.classList.add("active", "visible");
el.classList.remove("hidden");
el.classList.toggle("expanded");         // Add if absent, remove if present
el.classList.contains("active");        // → true or false
el.classList.replace("old", "new");

// Inline styles (only for dynamic JS-driven values)
el.style.backgroundColor = "#6366f1";
el.style.transform = "translateY(-4px)";

Creating & Inserting Elements

const newCard = document.createElement("div");
newCard.className = "card";
newCard.textContent = "New Item";

const container = document.querySelector(".grid");
container.appendChild(newCard);        // At end
container.prepend(newCard);            // At start
container.before(newCard);             // Before the container itself
existingEl.insertAdjacentElement("afterend", newCard);

// Template literals approach (simpler for complex HTML)
container.insertAdjacentHTML("beforeend", `
  <div class="card">
    <h3>New Card</h3>
    <p>Created dynamically</p>
  </div>
`);

Event Listeners

const btn = document.querySelector("#submit-btn");

// Add listener
btn.addEventListener("click", function(event) {
  event.preventDefault(); // Stop form submission / default action
  console.log("Button clicked!", event.target);
});

// Event delegation — handle events on parent for dynamic children
document.querySelector(".card-list").addEventListener("click", (e) => {
  if (e.target.matches(".card button")) {
    const card = e.target.closest(".card");
    card.remove();
  }
});

Removing Elements & Attributes

el.remove(); // Remove element from DOM

el.getAttribute("href");
el.setAttribute("href", "https://example.com");
el.removeAttribute("disabled");
