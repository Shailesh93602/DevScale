JavaScript Math

The Math object in JavaScript is a built-in object that allows you to perform mathematical tasks on numbers. Unlike other objects, Math is not a constructor; all its properties and methods are static.

1. Rounding and Precision

  • Math.round(x): Rounds to the nearest integer.
  • Math.ceil(x): Rounds up to the nearest integer.
  • Math.floor(x): Rounds down to the nearest integer.
  • Math.trunc(x): Returns the integer part of a number (ES6).

2. Pow and Sqrt

  • Math.pow(x, y): Returns x to the power of y. (Equivalent to x ** y).
  • Math.sqrt(x): Returns the square root of x.
  • Math.abs(x): Returns the absolute (positive) value of x.

3. Min and Max

Math.min(0, 150, 30, 20, -8, -200);  // Returns -200
  Math.max(0, 150, 30, 20, -8, -200);  // Returns 150

4. Random Numbers

Math.random() returns a floating-point, pseudo-random number in the range [0, 1) (inclusive of 0, but not 1).

// Example: Random number between 0 and 100
  Math.floor(Math.random() * 101);

5. Math Constants

  • Math.PI: 3.14159...
  • Math.E: Euler's number.
  • Math.SQRT2: Square root of 2.

Conclusion

The Math object is indispensable for any application involving calculations, whether it's for game development, data visualization, or even simple UI logic. Understanding how to use its built-in constants and static methods will save you from writing custom math utilities.
