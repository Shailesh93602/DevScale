JavaScript Regular Expressions (RegExp)

A Regular Expression is a sequence of characters that forms a search pattern. It's an incredibly powerful tool for pattern-matching and "search-and-replace" operations in text.

1. Syntax: Literals vs Constructor

// Literal syntax (preferred)
  let re = /pattern/flags;

  // Constructor syntax
  let re = new RegExp("pattern", "flags");

2. Common Methods

  • test(string): Returns true if the pattern is found in the string.
  • exec(string): Returns an array of matches or null.
  • match(regexp): String method that returns an array of matches.
  • replace(regexp, replacement): String method to search and replace.

3. Matching Basic Patterns

  • [abc]: Find any character between the brackets.
  • [0-9]: Find any digit between the brackets.
  • (x|y): Find any of the alternatives specified.

4. Meta-characters and Quantifiers

  • \d: Matches any digit.
  • \s: Matches any whitespace character.
  • \w: Matches any word character.
  • +: 1 or more occurrences.
  • *: 0 or more occurrences (quantifier).
  • ?: 0 or 1 occurrence (quantifier).
  • ^ and $: Anchors for the start and end of a string.

5. Regular Expression Flags

  • i: Ignore case (case-insensitive).
  • g: Global match (don't stop after the first match).
  • m: Multi-line match.

Conclusion

While regular expressions can seem cryptic at first, they are one of the most powerful tools in any developer's arsenal for tasks like form validation, data scraping, and text manipulation. Mastering even basic patterns will significantly improve your efficiency in handling strings.
