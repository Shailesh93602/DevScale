JavaScript Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can connect to the page.

1. Finding HTML Elements

To manipulate an element, you first need to find it using one of the following methods:

  • document.getElementById(id): Finds element by ID.
  • document.getElementsByClassName(name): Finds elements by class name.
  • document.querySelector(selector): Finds the first element matching a CSS selector.
  • document.querySelectorAll(selector): Finds all elements matching a CSS selector (returns a NodeList).

2. Changing HTML Content and Styles

const element = document.getElementById("header");
  element.innerHTML = "New Heading";      // Change text/HTML
  element.style.color = "blue";           // Change CSS style
  element.setAttribute("class", "main");  // Change attribute

3. Creating and Removing Elements

You can dynamically add or remove elements from the page using these core methods:

// Create a new div
  const newDiv = document.createElement("div");
  newDiv.innerHTML = "I am new!";

  // Add to the body
  document.body.appendChild(newDiv);

  // Remove an element
  const parent = document.getElementById("container");
  const child = document.getElementById("item");
  parent.removeChild(child);

4. Navigating the DOM Tree

The DOM is a tree of nodes. You can navigate between them using properties like parentNode, childNodes, firstChild, lastChild, nextSibling, and previousSibling.

Conclusion

The DOM is the bridge between your JavaScript code and the user's browser view. By mastering how to select, modify, and animate DOM elements, you can build fully interactive and dynamic user experiences without reaching for complex frameworks until necessary.
