JavaScript Image Maps

An image map is a regular image with clickable areas (hotspots). Using JavaScript, you can make these hotspots truly interactive by listening to events on specific portions of the image.

1. The Image Map HTML Structure

An image map requires an <img> tag with the usemap attribute and a <map> tag containing <area> tags.

<img src="planets.jpg" usemap="#planetmap" width="145" height="126">

<map name="planetmap">
  <area id="sun" shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm">
  <area id="mercury" shape="circle" coords="90,58,3" alt="Mercury" href="mercur.htm">
</map>

2. Interactive Image Maps with JavaScript

You can add event listeners to <area> tags just like any other HTML element. This allows you to trigger custom reactions based on where the user clicks inside an image.

const sunArea = document.getElementById("sun");

sunArea.addEventListener("mouseover", () => {
  console.log("Mouse is over the Sun!");
});

3. Dynamic Image Maps

Using JavaScript, you can dynamically change the coords or shape of an image map's hotspots, which is useful for responsive images where the size might change based on the screen width.

Conclusion

While modern web design often uses SVG for interactive graphics, the classic image map remains a useful tool for highlighting regions of an image and providing localized interaction. When combined with JavaScript events, it transforms a static graphic into a rich navigational experience.
