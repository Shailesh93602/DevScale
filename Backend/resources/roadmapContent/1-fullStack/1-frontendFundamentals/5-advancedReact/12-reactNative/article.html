React Native (Introduction)

React Native is a framework that allows you to build truly native mobile applications for iOS and Android using JavaScript and React. It’s developed by Meta and used by major companies like Instagram, Facebook, and Shopify.

1. Native Elements, Not the DOM

In React Native, you don’t use HTML tags like <div> or <p>. Instead, you use native components provided by the framework that map directly to the platform’s original UI components.

  • <View />: Like a <div>, the basic building block of mobile UI.
  • <Text />: Like a <p>, used to display strings.
  • <Image />: Like an <img>, used for displaying assets.

2. Styling with StyleSheet

React Native doesn't use standard CSS files. Instead, it uses a StyleSheet API that accepts a JavaScript object where keys are very similar to CSS properties (in camelCase).

3. Expo vs. React Native CLI

  • Expo: A set of tools and services that makes React Native development much easier to start and manage for beginners and smaller apps.
  • CLI: The standard way to build a mobile app with more control over the native platform configuration (Xcode/Android Studio).

Conclusion

React Native is the industry-standard framework for building cross-platform mobile apps with a high-performance, native feel. By leveraging your React knowledge, you can extend your skills from web development to mobile development with minimal friction and maximum reach.
