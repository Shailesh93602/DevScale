The Vue Instance (Application)

Every Vue application starts by creating a new **application instance** with the createApp function. This instance is where you define your app's data, methods, and configurations before attaching it to the DOM.

1. Creating the App Instance

To create a Vue app, you pass your single root component (the object representing your app) to createApp(). This creates an instance that has its own isolated scope.

import { createApp } from 'vue'

const app = createApp({
  /* root component options */
})

2. Mounting the Instance

After creating the instance, you must "mount" it to a real DOM element using the mount() method. This tells Vue where on your webpage it should render the application.

app.mount('#app')

3. The Data Object

At its heart, the Vue instance tracks state using a data function. Any values returned from this function are automatically made reactive by the framework.

createApp({
  data() {
    return {
      message: 'Hello Vue!'
    }
  }
}).mount('#app')

Conclusion

The Vue instance is the starting point of every interactive piece of your application. By mastering how to create, configure, and mount your application instances, you lay the groundwork for building complex and reactive user interfaces that always stay in sync with your data.
