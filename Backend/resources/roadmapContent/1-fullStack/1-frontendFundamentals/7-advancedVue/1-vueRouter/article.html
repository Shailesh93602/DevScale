Vue Router

Vue Router is the official router for Vue.js. It is deeply integrated with Vue.js core to make building Single Page Applications (SPAs) with Vue a breeze. It allows you to map your application's URLs to components, enabling smooth and flickering-free navigation.

1. Core Components

  • <router-link>: Use this instead of standard anchor tags (<a>) for internal navigation. It automatically handles URL updates without page reloads.
  • <router-view>: This is a functional component that renders the component for the current route as defined in your configuration.

2. Basic Configuration

To set up routing, you define an array of route objects, each mapping a URL path to a component. You then create a router instance and pass it to your main Vue application.

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

app.use(router)

3. Dynamic Route Matching

You can use parameters in your route paths to create dynamic user-specific pages. For example, /user/:id will match any URL like /user/123 or /user/john.

Conclusion

Vue Router is an essential piece of any professional Vue.js application. By mastering how to manage navigation, map URLs to components, and use dynamic routes, you can provide your users with the fast and seamless multi-page experience that characterizes modern single-page web applications.
