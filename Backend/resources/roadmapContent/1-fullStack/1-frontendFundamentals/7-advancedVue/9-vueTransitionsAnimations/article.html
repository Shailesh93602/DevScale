Vue Transitions and Animations

Vue.js provides a built-in <Transition> component that makes it easy to add enter and leave transitions for elements or components when they are shown or hidden with v-if or v-show.

1. Transition CSS Classes

When an element wrapped in <Transition name="fade"> enters or leaves the DOM, Vue automatically adds and removes specific CSS classes at different stages. By defining styles for these classes, you can create smooth and customizable animations.

  • v-enter-from: Initial state of the element before its added.
  • v-enter-active: Applied during the enter transition.
  • v-enter-to: Ending state of the element's enter transition.
  • v-leave-active: Applied during the leave transition.

2. Animating Lists (TransitionGroup)

To animate multiple items in a list (e.g. from v-for), you should use the <TransitionGroup> component. This works similarly to <Transition> but supports animating several elements at once.
