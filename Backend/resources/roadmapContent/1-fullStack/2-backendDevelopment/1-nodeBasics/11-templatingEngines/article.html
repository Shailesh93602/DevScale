Templating Engines

To serve dynamic HTML from your server, you need a **Templating Engine**. These tools allow you to embed JavaScript directly into your HTML, making it easy to show a user's name, display a list of products, or render content based on a database query.

1. Popular Directives (EJS, Pug, Handlebars)

  • EJS (Embedded JavaScript): The most popular and standard templating engine, as it uses standard HTML syntax with JavaScript tags (<%= user.name %>).
  • Pug: Uses a specialized, indented syntax that eliminates the need for closing tags, resulting in very clean and short code.
  • Handlebars: Provides a "minimalist" approach that focuses on keeping your logic out of your templates as much as possible.

2. Setting Up Express

To use a templating engine, you simply tell Express which one you've installed. For example, to use **EJS**, you would add app.set('view engine', 'ejs') to your project configuration.

3. Professional Rendering

Instead of using res.send(), you used res.render() to combine your template with the specific data for that request. This is the industry standard for server-side website generation.

Conclusion
