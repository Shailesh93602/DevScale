HTTP Module

To turn your Node.js application into a web server, you must use the **HTTP Module**. This allows you to listen for incoming network requests and send back professional, structured responses to your users.

1. Creating a Server

The http.createServer method is the foundation of almost every Node.js web server. It provides a callback that runs every time a user visits your site.

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/html');
  res.end('<h1>Hello from Node.js!</h1>');
});

server.listen(3000, () => {
  console.log('Server running on port 3000');
});

2. Request and Response (req and res)

  • req (Request): Contains all information from the user (URL, headers, query parameters, data).
  • res (Response): Is used to send data back to the user (status codes, headers, and the final HTML/JSON content).

3. Routing Basics

By using the req.url property, you can build a simple "router" to show different pages to your users based on the path they requested (e.g. /about or /contact).
