Static Files in Express

To build a professional web application, you need to serve more than just raw HTML. Your site will need images, CSS styles, and client-side JavaScript—collectively known as **Static Files**.

1. express.static() Middleware

Express provides a built-in middleware called express.static() that automatically serves files from a specific directory on your server. Professional developers usually create a folder called public or dist to hold these assets.

app.use(express.static('public'));

2. How it Works

If you have a file at public/css/style.css, it will be accessible in your browser at http://localhost:3000/css/style.css. By using this middleware, your server can handle hundreds of requests for images and styles efficiently without you having to write any custom route handlers.
