Introduction to React & JSX

React is a JavaScript library for building user interfaces, created by Meta (Facebook). It uses a component-based architecture and a virtual DOM for efficient rendering.

Why React?

  • Component-Based — Build encapsulated components, then compose them into complex UIs.
  • Virtual DOM — React maintains a lightweight copy of the DOM; only changed parts re-render.
  • Declarative — Describe what the UI should look like for a given state; React handles the DOM updates.
  • Huge Ecosystem — React Native (mobile), Next.js (SSR), thousands of libraries.

JSX — JavaScript + HTML

JSX lets you write HTML-like syntax directly inside JavaScript. Babel transforms JSX into React.createElement() calls.

// JSX
const element = <h1 className="title">Hello, {name}!</h1>;

// What Babel compiles it to:
const element = React.createElement("h1", { className: "title" }, "Hello, ", name, "!");

JSX Rules

// 1. Must return a single root element (or Fragment)
return (
  <>
    <h1>Title</h1>
    <p>Paragraph</p>
  </>
);

// 2. Use className instead of class
<div className="container">

// 3. Self-close empty elements
<img src="logo.png" alt="Logo" />
<br />

// 4. JavaScript expressions in curly braces
<p>{2 + 2}</p>
<p>{isLoggedIn ? "Welcome!" : "Please log in"}</p>

// 5. camelCase for event handlers and attributes
<button onClick={handleClick} tabIndex={0}>Click me</button>

Your First Component

// A functional component is just a function that returns JSX
function WelcomeCard({ name, role }: { name: string; role: string }) {
  return (
    <div className="card">
      <h2>Welcome, {name}!</h2>
      <p>Role: <strong>{role}</strong></p>
    </div>
  );
}

// Using the component
function App() {
  return <WelcomeCard name="Alice" role="Admin" />;
}

Rendering Lists

const fruits = ["Apple", "Banana", "Cherry"];

function FruitList() {
  return (
    <ul>
      {fruits.map((fruit) => (
        <li key={fruit}>{fruit}</li> // Always add key prop!
      ))}
    </ul>
  );
}

Conditional Rendering

function Greeting({ isLoggedIn }: { isLoggedIn: boolean }) {
  return (
    <div>
      {isLoggedIn ? (
        <p>Welcome back!</p>
      ) : (
        <p>Please sign in.</p>
      )}
      {isLoggedIn && <button>Log out</button>} {/* Short-circuit */}
    </div>
  );
}
