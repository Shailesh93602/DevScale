React Components, Props & Composition

Components are the building blocks of every React UI. Props allow data to flow downward from parent to child components.

Functional Components

// Simple functional component
function Button({ label, onClick, variant = "primary" }: {
  label: string;
  onClick: () => void;
  variant?: "primary" | "secondary";
}) {
  return (
    <button
      className={`btn btn-${variant}`}
      onClick={onClick}
    >
      {label}
    </button>
  );
}

// Usage
<Button label="Save" onClick={() => save()} />
<Button label="Cancel" onClick={() => cancel()} variant="secondary" />

Props Rules

  • Props are read-only — a component must never modify its own props.
  • Props flow one-way — parent to child (downward).
  • Any JavaScript value can be a prop: strings, numbers, arrays, objects, functions, JSX.

The children Prop

function Card({ title, children }: { title: string; children: React.ReactNode }) {
  return (
    <div className="card">
      <h3 className="card-title">{title}</h3>
      <div className="card-body">{children}</div>
    </div>
  );
}

// Usage — any JSX nested inside becomes `children`
<Card title="User Info">
  <p>Name: Alice</p>
  <p>Role: Admin</p>
</Card>

Component Composition Patterns

// Compound Components Pattern
function Tabs({ children }: { children: React.ReactNode }) {
  const [active, setActive] = React.useState(0);
  return <div className="tabs">{children}</div>;
}

Tabs.Tab = function Tab({ label }: { label: string }) {
  return <button className="tab">{label}</button>;
};

// Usage
<Tabs>
  <Tabs.Tab label="Overview" />
  <Tabs.Tab label="Details" />
</Tabs>

Passing Functions as Props (Lifting State Up)

// Parent owns the state; passes handler down to child
function SearchPage() {
  const [query, setQuery] = React.useState("");

  return (
    <div>
      <SearchInput
        value={query}
        onChange={(val) => setQuery(val)} // Pass handler as prop
      />
      <SearchResults query={query} />
    </div>
  );
}

function SearchInput({ value, onChange }: { value: string; onChange: (v: string) => void }) {
  return (
    <input
      value={value}
      onChange={(e) => onChange(e.target.value)}
      placeholder="Search..."
    />
  );
}
