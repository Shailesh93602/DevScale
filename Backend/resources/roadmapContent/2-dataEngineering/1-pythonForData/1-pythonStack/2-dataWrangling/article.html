Data Wrangling with Pandas

Data wrangling (also called data munging) is the process of cleaning and transforming raw data into a usable format. In practice, ~80% of a data engineer's time is spent here. Pandas is the essential Python library for this work.

Loading Data

import pandas as pd

df = pd.read_csv('data.csv')
df = pd.read_json('data.json')
df = pd.read_parquet('data.parquet')
df = pd.read_sql("SELECT * FROM orders", con=engine)

Exploring a Dataset

df.shape          # (rows, columns)
df.dtypes         # column data types
df.info()         # non-null counts + dtypes
df.describe()     # mean, std, min, max
df.head(5)        # first 5 rows
df.isnull().sum() # missing values per column

Handling Missing Values

# Drop rows with any NaN
df.dropna(inplace=True)

# Drop only if specific column is NaN
df.dropna(subset=['email'], inplace=True)

# Fill with a specific value
df['age'].fillna(df['age'].median(), inplace=True)

# Forward fill (time series)
df['price'].ffill(inplace=True)

Filtering and Selecting

# Boolean filter
adults = df[df['age'] >= 18]

# Multiple conditions
active_users = df[(df['status'] == 'active') & (df['age'] > 21)]

# Select columns
subset = df[['name', 'email', 'age']]

# iloc — by position; loc — by label
df.iloc[0:5, 1:3]
df.loc[df['country'] == 'India', 'city']

Transforming Data

# Apply a function
df['name'] = df['name'].str.strip().str.title()
df['revenue'] = df['price'] * df['quantity']

# Rename columns
df.rename(columns={'prc': 'price', 'qty': 'quantity'}, inplace=True)

# Change types
df['date'] = pd.to_datetime(df['date'])
df['price'] = df['price'].astype(float)

Grouping and Aggregation

summary = df.groupby('category').agg(
    total_revenue=('revenue', 'sum'),
    avg_price=('price', 'mean'),
    order_count=('id', 'count')
).reset_index()

# Pivot table
pivot = df.pivot_table(values='sales', index='region', columns='month', aggfunc='sum')
