Data Visualisation with Python

Data visualisation turns raw numbers into actionable insights. Python's ecosystem — Matplotlib (low-level), Seaborn (statistical), and Plotly (interactive) — covers every use case from quick EDA to production dashboards.

Matplotlib — The Foundation

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.figure(figsize=(10, 4))
plt.plot(x, y, color='royalblue', linewidth=2, label='sin(x)')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Sine Wave')
plt.legend()
plt.grid(alpha=0.3)
plt.tight_layout()
plt.savefig('sine.png', dpi=150)
plt.show()

Seaborn — Statistical Visualisation

import seaborn as sns
import pandas as pd

df = sns.load_dataset('tips')

# Distribution plot
sns.histplot(df['total_bill'], kde=True)

# Categorical comparison
sns.boxplot(data=df, x='day', y='total_bill', hue='sex')

# Correlation heatmap
sns.heatmap(df.corr(numeric_only=True), annot=True, cmap='coolwarm')

# Pair plot — visualise all column relationships
sns.pairplot(df, hue='sex')

Plotly — Interactive Charts

import plotly.express as px

df = px.data.gapminder()

# Animated scatter
fig = px.scatter(df, x='gdpPercap', y='lifeExp',
                 size='pop', color='continent',
                 hover_name='country',
                 animation_frame='year',
                 log_x=True)
fig.show()

# Bar chart
fig = px.bar(df[df.year==2007], x='country', y='gdpPercap',
             color='continent', title='GDP per Capita 2007')
fig.show()

Subplots

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].plot(x, np.sin(x))
axes[0].set_title('Sine')
axes[1].plot(x, np.cos(x), color='orange')
axes[1].set_title('Cosine')
plt.tight_layout()

Best Practices

  • Always label axes and add a title.
  • Use colorblind-friendly palettes (e.g., sns.color_palette('colorblind')).
  • Export at high DPI (150–300) for reports.
  • Use Plotly for interactive dashboards shared with stakeholders.
  • Keep charts simple — one key message per chart.
