SQL Window Functions

Window functions perform calculations across a set of rows related to the current row — without collapsing rows like GROUP BY does. They are essential for advanced analytics: rankings, running totals, moving averages, and period-over-period comparisons.

Syntax

function_name(expression)
  OVER (
    [PARTITION BY column]
    [ORDER BY column]
    [ROWS/RANGE frame]
  )

Ranking Functions

SELECT
  employee_id,
  department,
  salary,
  ROW_NUMBER()  OVER (PARTITION BY department ORDER BY salary DESC) AS row_num,
  RANK()        OVER (PARTITION BY department ORDER BY salary DESC) AS rank,
  DENSE_RANK()  OVER (PARTITION BY department ORDER BY salary DESC) AS dense_rank,
  NTILE(4)      OVER (ORDER BY salary DESC) AS quartile
FROM employees;
  • ROW_NUMBER — unique sequential number, no ties
  • RANK — ties get same rank, next rank skips (1,1,3)
  • DENSE_RANK — ties get same rank, no skipping (1,1,2)
  • NTILE(n) — divides rows into n equal buckets

Lead & Lag — Period Comparisons

SELECT
  month,
  revenue,
  LAG(revenue, 1)  OVER (ORDER BY month) AS prev_month_revenue,
  LEAD(revenue, 1) OVER (ORDER BY month) AS next_month_revenue,
  revenue - LAG(revenue, 1) OVER (ORDER BY month) AS mom_change
FROM monthly_sales;

Running Totals & Moving Averages

SELECT
  date,
  amount,
  SUM(amount)  OVER (ORDER BY date ROWS UNBOUNDED PRECEDING) AS running_total,
  AVG(amount)  OVER (ORDER BY date ROWS BETWEEN 6 PRECEDING AND CURRENT ROW) AS ma_7day
FROM daily_revenue;

First & Last Value

SELECT
  product_id,
  sale_date,
  price,
  FIRST_VALUE(price) OVER (PARTITION BY product_id ORDER BY sale_date) AS first_price,
  LAST_VALUE(price)  OVER (PARTITION BY product_id ORDER BY sale_date
                           ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING) AS last_price
FROM sales;

Percentile Functions

SELECT
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY salary) AS median_salary,
  PERCENTILE_CONT(0.9) WITHIN GROUP (ORDER BY salary) AS p90_salary
FROM employees;

Common Use Cases

  • Find the top N rows per group (ROW_NUMBER + WHERE row_num = 1)
  • Month-over-month growth (LAG)
  • 7-day moving average (AVG ROWS frame)
  • Cumulative sum of revenue (SUM UNBOUNDED PRECEDING)
  • Rank employees by salary within department (DENSE_RANK)
