Data Visualization: Matplotlib & Seaborn

Before training any Machine Learning model, you must understand your dataset. Hidden outliers distort model training, perfectly correlated features add redundancy, and skewed distributions violate algorithm assumptions. Data visualization is the fastest tool for discovering these patterns — it turns thousands of numbers into instantly readable images that reveal what statistical summaries miss.

Matplotlib — The Foundation

Matplotlib is the foundational Python visualization library. Almost every other library (Seaborn, Pandas .plot()) is built on top of it. Understanding its Figure and Axes model unlocks full control:

import matplotlib.pyplot as plt
import numpy as np

# ─── Basic structure: Figure (window) contains Axes (individual plots) ─────
fig, axes = plt.subplots(nrows=1, ncols=2, figsize=(12, 5))

# Left plot: line chart
x = np.linspace(0, 2 * np.pi, 200)
axes[0].plot(x, np.sin(x), color='steelblue', linewidth=2, label='sin(x)')
axes[0].plot(x, np.cos(x), color='coral', linewidth=2, linestyle='--', label='cos(x)')
axes[0].set_title('Trigonometric Functions', fontsize=14)
axes[0].set_xlabel('x')
axes[0].set_ylabel('y')
axes[0].legend()
axes[0].grid(alpha=0.3)

# Right plot: scatter chart
np.random.seed(42)
x = np.random.randn(100)
y = 2 * x + np.random.randn(100) * 0.5  # Linear relationship with noise
scatter = axes[1].scatter(x, y, c=y, cmap='viridis', alpha=0.7, edgecolors='white', linewidth=0.5)
axes[1].set_title('Correlated Variables', fontsize=14)
plt.colorbar(scatter, ax=axes[1], label='y value')

plt.tight_layout()
plt.savefig('analysis.png', dpi=150, bbox_inches='tight')
plt.show()

Seaborn — Statistical Visualization

Seaborn is a higher-level library built on Matplotlib, designed specifically for statistical graphics. It integrates directly with Pandas DataFrames, automatically handles grouping and aggregation, and produces publication-quality plots with minimal code:

import seaborn as sns
import pandas as pd
import matplotlib.pyplot as plt

# Use the built-in Titanic dataset for examples
df = sns.load_dataset('titanic')

# Set a clean global theme
sns.set_theme(style='whitegrid', palette='husl', font_scale=1.1)

Histplot — Distribution of a Single Variable

# Is age normally distributed? Are there outlier ages?
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

sns.histplot(df['age'].dropna(), kde=True, bins=30, ax=ax1, color='steelblue')
ax1.set_title('Age Distribution of Passengers')

# Group by survival — did age affect survival odds?
sns.histplot(data=df, x='age', hue='survived', kde=True, bins=25, ax=ax2)
ax2.set_title('Age Distribution by Survival')
plt.tight_layout()
plt.show()

Boxplot — Comparing Groups & Finding Outliers

# Did ticket class (1st/2nd/3rd) correlate with age? Are there outliers?
plt.figure(figsize=(8, 5))
sns.boxplot(data=df, x='pclass', y='age', hue='survived',
            palette={'0': '#ef4444', '1': '#22c55e'})
plt.title('Passenger Age by Class and Survival')
plt.xlabel('Ticket Class (1=First, 2=Second, 3=Third)')
plt.ylabel('Age')
plt.show()
# Dots outside the whiskers = statistical outliers (age > Q3 + 1.5*IQR)

Heatmap — Correlation Matrix (Critical for Feature Selection)

# Find features that are highly correlated with each other
# Highly correlated features (|r| > 0.8) are often redundant — drop one!
numeric_df = df[['survived', 'pclass', 'age', 'sibsp', 'parch', 'fare']].copy()
correlation_matrix = numeric_df.corr()

plt.figure(figsize=(8, 6))
sns.heatmap(
    correlation_matrix,
    annot=True,          # Show correlation values inside cells
    fmt='.2f',           # 2 decimal places
    cmap='coolwarm',     # Blue = negative, Red = positive correlation
    center=0,            # Center the color map at zero
    square=True,
    linewidths=0.5,
    cbar_kws={'shrink': 0.8}
)
plt.title('Feature Correlation Matrix')
plt.tight_layout()
plt.show()
# Tip: If two features have |r| > 0.85, they carry nearly identical information

Pairplot — The EDA Power Tool

# One function plots every numeric column against every other column
# + distribution histograms on the diagonal — best first EDA step
sns.pairplot(
    df[['survived', 'pclass', 'age', 'fare']].dropna(),
    hue='survived',          # Color by target variable
    diag_kind='hist',        # Diagonal: histograms (or 'kde')
    plot_kws={'alpha': 0.5, 'edgecolor': 'none'},
    palette={0: '#ef4444', 1: '#22c55e'}
)
plt.suptitle('Pairplot — Titanic Dataset', y=1.02, fontsize=14)
plt.show()

Barplot & Countplot — Categorical Analysis

# What percentage of each class survived? (barplot shows mean + 95% CI)
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

sns.barplot(data=df, x='pclass', y='survived', ax=ax1, palette='Blues_d', errorbar='ci')
ax1.set_title('Survival Rate by Class')
ax1.set_ylabel('Proportion Surviving')

# How many passengers in each embarkation port?
sns.countplot(data=df, x='embarked', hue='survived', ax=ax2, palette='Set2')
ax2.set_title('Embarked Port vs Survival')
plt.tight_layout()
plt.show()

A Practical EDA Workflow

  1. Shape & types: df.info(), df.describe() — understand columns and basic statistics
  2. Missing values: df.isnull().sum() + sns.heatmap(df.isnull())
  3. Target distribution: df['label'].value_counts(normalize=True) — detect class imbalance
  4. Univariate: sns.histplot for numeric, sns.countplot for categorical
  5. Correlation: sns.heatmap(df.corr()) — find redundant or predictive features
  6. Bivariate: sns.pairplot — spot linear relationships across all feature pairs
  7. Outliers: sns.boxplot — identify and decide whether to remove or cap outliers

Best Practices

  • Always do EDA before modeling — "garbage in, garbage out" is especially true in ML. Visualizing data takes 30 minutes; debugging a biased model takes days.
  • Save plots to files (plt.savefig()) in your ML pipeline so they are reproducible artifacts, not ephemeral Jupyter outputs.
  • Use sns.set_theme(style='whitegrid') at the top of every notebook for consistent, readable styling.
  • When features have very different scales (age: 0–80, salary: 0–500,000), standardize before plotting correlations to avoid misleading visual patterns.
