Feature Engineering & Data Preprocessing

"Garbage in, garbage out" — no machine learning model can compensate for poorly prepared data. Feature engineering is often cited as the highest-leverage activity in applied ML, frequently contributing more to model performance than the choice of algorithm itself.

Handling Missing Values

import pandas as pd
from sklearn.impute import SimpleImputer, KNNImputer

df = pd.read_csv("housing.csv")

# Understand the missingness pattern first
print(df.isnull().sum())
print(df.isnull().mean() * 100)  # % missing per column

# Strategy 1: Simple imputation
# Numerical: use median (robust to outliers)

Encoding Categorical Variables

from sklearn.preprocessing import OneHotEncoder, OrdinalEncoder
import pandas as pd

# One-Hot Encoding — for nominal categories (no inherent order)
# 'color': ['red', 'blue', 'green'] → [1,0,0], [0,1,0], [0,0,1]
# Use when: <= 10 unique values, tree-based models
encoder = OneHotEncoder(sparse_output=False, handle_unknown='ignore')
color_encoded = encoder.fit_transform(df[['color']])

# Ordinal Encoding — for ordered categories
# 'size': ['small', 'medium', 'large'] → [0, 1, 2]
ordinal = OrdinalEncoder(categories=[['small', 'medium', 'large']])
df['size_encoded'] = ordinal.fit_transform(df[['size']])

# Target Encoding — for high cardinality categoricals (e.g., zip codes)
# Replaces each category with the mean target value for that category
# ⚠️ Always use cross-validation to avoid target leakage
df['zip_encoded'] = df.groupby('zip_code')['price'].transform('mean')

Building Sklearn Pipelines

from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer

# Define transformations for different column types
numeric_transformer = Pipeline(steps=[
    ('imputer', SimpleImputer(strategy='median')),
    ('scaler', StandardScaler())
])
categorical_transformer = Pipeline(steps=[
    ('imputer', SimpleImputer(strategy='most_frequent')),
    ('encoder', OneHotEncoder(handle_unknown='ignore'))
])

preprocessor = ColumnTransformer(transformers=[
    ('num', numeric_transformer, ['age', 'income', 'sqft']),
    ('cat', categorical_transformer, ['city', 'property_type'])
])

# The Pipeline guarantees no data leakage — all preprocessing is fit on training data
full_pipeline = Pipeline(steps=[
    ('preprocessor', preprocessor),
