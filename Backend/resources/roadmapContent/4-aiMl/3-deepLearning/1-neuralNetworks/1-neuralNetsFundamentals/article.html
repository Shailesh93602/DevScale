Neural Networks & Backpropagation

Neural networks are computational models loosely inspired by biological brains. They learn by adjusting millions of numerical parameters through a process called backpropagation.

The Neuron

# A neuron: weighted sum of inputs → activation function → output
# output = activation(w₁x₁ + w₂x₂ + ... wₙxₙ + bias)

# Activation functions:
# ReLU(x) = max(0, x)           — default for hidden layers
# Sigmoid(x) = 1/(1+e^(-x))    — binary classification output (0-1)
# Softmax(x)                    — multiclass output (probabilities sum to 1)
# Tanh(x)                       — alternative hidden layer activation

Building Neural Networks with PyTorch

import torch
import torch.nn as nn

class ChurnPredictor(nn.Module):
    def __init__(self, input_dim: int):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, 128),
            nn.BatchNorm1d(128),   # Normalise activations for stable training
            nn.ReLU(),
            nn.Dropout(0.3),       # Randomly zero 30% of neurons (regularisation)
            nn.Linear(128, 64),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(64, 1),
            nn.Sigmoid()           # Binary output
        )

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.net(x)

optimizer = torch.optim.Adam(model.parameters(), lr=1e-3, weight_decay=1e-4)

for epoch in range(100):
    model.train()                     # Enable dropout + batch norm training mode
    optimizer.zero_grad()             # Reset gradients
    predictions = model(X_train_t)    # Forward pass
    loss = criterion(predictions, y_train_t)  # Compute loss
    loss.backward()                   # Backpropagation
    optimizer.step()                  # Update weights

Backpropagation Intuition

  • Forward pass — compute predictions and loss.
  • loss.backward() — uses chain rule to compute ∂loss/∂w for every weight.
  • optimizer.step() — moves weights slightly in the direction that reduces loss.
  • Repeat millions of times → weights converge to minimise loss.
