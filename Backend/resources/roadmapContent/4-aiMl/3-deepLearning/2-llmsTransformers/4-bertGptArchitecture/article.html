BERT, GPT & Foundation Model Architecture

Modern NLP is dominated by two architectural paradigms: BERT and GPT. Both are transformer-based and trained on massive text corpora, but their pre-training objectives determine what they're best at and how they should be used for downstream tasks.

BERT: Encoder-Only, Bidirectional Understanding

BERT (Bidirectional Encoder Representations from Transformers) pre-trains on two tasks: Masked Language Modeling (predict randomly masked tokens using context from both directions) and Next Sentence Prediction:

from transformers import BertTokenizer, BertForSequenceClassification
import torch

# BERT for text classification (fine-tuning)
tokenizer = BertTokenizer.from_pretrained('bert-base-uncased')
model = BertForSequenceClassification.from_pretrained(
    'bert-base-uncased',
    num_labels=2   # Binary classification (e.g., positive/negative sentiment)
)

# Tokenize input — BERT requires special tokens
text = "The product quality exceeded my expectations!"
inputs = tokenizer(
    text,
    return_tensors='pt',
    max_length=512,
    padding='max_length',
    truncation=True
)
# inputs contains: input_ids, attention_mask, token_type_ids

# Fine-tuning BERT: only train the classification head for a few epochs
# The BERT body already contains rich language understanding from pretraining
optimizer = torch.optim.AdamW(model.parameters(), lr=2e-5)  # Very small LR for fine-tuning

outputs = model(**inputs, labels=torch.tensor([1]))
loss = outputs.loss

# BERT excels at: classification, NER, question answering, semantic similarity
# Cannot generate text — it's encoder-only

GPT: Decoder-Only, Autoregressive Generation

from transformers import GPT2LMHeadModel, GPT2Tokenizer

tokenizer = GPT2Tokenizer.from_pretrained('gpt2')
model = GPT2LMHeadModel.from_pretrained('gpt2')

# GPT generates text autoregressively: predict next token, append, repeat
input_text = "The future of artificial intelligence is"
inputs = tokenizer(input_text, return_tensors='pt')

# Generation strategies
output = model.generate(
    inputs['input_ids'],
    max_new_tokens=100,
    do_sample=True,        # Sample from distribution (diverse output)
    temperature=0.7,       # Lower = more conservative, Higher = more creative
    top_p=0.9,             # Nucleus sampling: sample from top 90% probability mass
    top_k=50,              # Only sample from top 50 most likely tokens
    repetition_penalty=1.2 # Penalize repeating tokens
)

generated_text = tokenizer.decode(output[0], skip_special_tokens=True)

T5: Encoder-Decoder, Text-to-Text

from transformers import T5ForConditionalGeneration, T5Tokenizer

# T5's elegant insight: ALL NLP tasks can be framed as text-to-text
# Translation:    "translate English to French: Hello world" → "Bonjour le monde"
# Summarization:  "summarize: [long article text]" → "Brief summary"
# Classification: "sentiment: This movie was amazing" → "positive"
# QA:             "question: What is the capital? context: France is..." → "Paris"

model = T5ForConditionalGeneration.from_pretrained('t5-base')
tokenizer = T5Tokenizer.from_pretrained('t5-base')

# Summarization example
article = "Amazon reported record profits... [long text]"
input_text = f"summarize: {article}"
inputs = tokenizer(input_text, return_tensors='pt', max_length=512, truncation=True)
summary_ids = model.generate(inputs['input_ids'], max_new_tokens=150, num_beams=4)
print(tokenizer.decode(summary_ids[0], skip_special_tokens=True))

Choosing the Right Architecture

Task                      Best Architecture       Examples
─────────────────────────────────────────────────────────
Text classification        Encoder-only (BERT)    Sentiment, spam, intent
Named Entity Recognition   Encoder-only (BERT)    POS tagging, NER
Semantic similarity        Encoder-only (BERT)    STS, duplicate detection
Text generation            Decoder-only (GPT)     Chatbots, code completion
Translation/Summarization  Encoder-Decoder (T5)   MT, abstractive summarization
Open-ended generation      Decoder-only (GPT)     Creative writing, QA with context
