Prompt Engineering & LLM Evaluation

Prompt engineering is the art of communicating with large language models effectively. A well-crafted prompt can transform a mediocre response into an expert-quality answer — often without any fine-tuning required. In production, systematic prompt engineering is cheaper, faster, and more maintainable than fine-tuning for many use cases.

Core Prompting Techniques

from openai import OpenAI
client = OpenAI()

# Zero-Shot: no examples, just task description
zero_shot = """Classify the following customer review as POSITIVE, NEGATIVE, or NEUTRAL.
Reply with only the classification label.

Review: "Delivery was fast but the product quality was disappointing."
Classification:"""

# Few-Shot: provide labeled examples before the actual input
few_shot = """Classify customer reviews. Reply with POSITIVE, NEGATIVE, or NEUTRAL only.

Review: "Absolutely love this product, exceeded expectations!"
Classification: POSITIVE

Review: "Arrived damaged and customer service was unhelpful."
Classification: NEGATIVE

Review: "It works as described, nothing special."
Classification: NEUTRAL

Review: "Delivery was fast but the product quality was disappointing."
Classification:"""

# Chain-of-Thought: ask the model to reason step-by-step before answering
# Dramatically improves accuracy on complex reasoning tasks
chain_of_thought = """Solve this problem step by step, then give the final answer.

Problem: A store sells shirts for $25 each. Buy 3 get 1 free. What do 5 shirts cost?

Steps:"""

response = client.chat.completions.create(
    model="gpt-4-turbo",
    messages=[{"role": "user", "content": chain_of_thought}]
)

Structured Output with JSON Mode

from pydantic import BaseModel
from openai import OpenAI
import json

# JSON mode: forces the model to output valid JSON
# Use for: extracting structured data from unstructured text

class SentimentAnalysis(BaseModel):
    sentiment: str          # "positive" | "negative" | "neutral"
    confidence: float       # 0.0 to 1.0
    key_phrases: list[str]  # Phrases that drove the sentiment

response = client.chat.completions.create(
    model="gpt-4-turbo",
    response_format={"type": "json_object"},
    messages=[{
        "role": "system",
        "content": f"You analyze customer reviews. Respond with JSON matching this schema: {SentimentAnalysis.schema()}"
    }, {
        "role": "user",
        "content": "Review: 'The battery dies too quickly but camera quality is stunning.'"
    }]
)

result = SentimentAnalysis(**json.loads(response.choices[0].message.content))
