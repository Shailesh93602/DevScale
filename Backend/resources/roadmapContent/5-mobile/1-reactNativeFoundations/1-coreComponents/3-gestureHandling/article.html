Gesture Handling & Touch Interactions

React Native's default JS-thread touch system creates janky interactions because touch events must cross the JS/native bridge before registering. React Native Gesture Handler (RNGH) solves this by running gesture recognition entirely on the UI thread — producing silky-smooth interactions that feel truly native.

Why Gesture Handler Over TouchableOpacity

TouchableOpacity (old approach):
  Touch → Native bridge → JS thread → JS processes → Native bridge → UI update
  Delay: 16-50ms per step — noticeable lag during heavy JS work

React Native Gesture Handler:
  Touch → Native gesture recognizer on UI thread → Immediate UI response
  JS thread: only notified AFTER the gesture is already handled visually
  
Rule: For any interactive element that demands instant visual feedback
       (buttons, sliders, drag handles) — always use RNGH over Touchable*.

Swipe-to-Delete with PanGesture

import { GestureDetector, Gesture } from 'react-native-gesture-handler';
import Animated, {
  useSharedValue, useAnimatedStyle, withSpring, runOnJS
} from 'react-native-reanimated';

const SwipeableRow = ({ item, onDelete }) => {
  const translateX = useSharedValue(0);
  const SWIPE_THRESHOLD = -80;

  const panGesture = Gesture.Pan()
    .onUpdate((event) => {
      // Only allow left swipe (negative X)
      translateX.value = Math.min(0, event.translationX);
    })
    .onEnd((event) => {
      if (event.translationX < SWIPE_THRESHOLD) {
        // Swiped far enough → snap to delete position
        translateX.value = withSpring(-120);
      } else {
        // Not far enough → snap back
        translateX.value = withSpring(0);
      }
    });

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ translateX: translateX.value }],
  }));

  return (
    <View style={styles.container}>
      {/* Delete button revealed underneath */}
      <View style={styles.deleteButton}>
        <Text onPress={() => runOnJS(onDelete)(item.id)}>Delete</Text>
      </View>
      {/* Swipeable row on top */}
      <GestureDetector gesture={panGesture}>
        <Animated.View style={[styles.row, animatedStyle]}>
          <Text>{item.title}</Text>
        </Animated.View>
      </GestureDetector>
    </View>
  );
};

Pinch-to-Zoom

import { Gesture, GestureDetector } from 'react-native-gesture-handler';
import Animated, { useSharedValue, useAnimatedStyle } from 'react-native-reanimated';

const ZoomableImage = ({ source }) => {
  const scale = useSharedValue(1);
  const savedScale = useSharedValue(1);

  const pinchGesture = Gesture.Pinch()
    .onUpdate((event) => {
      // Multiply current saved scale by ongoing pinch scale
      scale.value = savedScale.value * event.scale;
    })
    .onEnd(() => {
      // Clamp scale between 1x and 4x, then save
      savedScale.value = Math.min(Math.max(scale.value, 1), 4);
      scale.value = savedScale.value;
    });

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ scale: scale.value }],
  }));

  return (
    <GestureDetector gesture={pinchGesture}>
      <Animated.Image source={source} style={[styles.image, animatedStyle]} />
    </GestureDetector>
  );
};
