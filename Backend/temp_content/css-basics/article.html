CSS Basics: Styling Your Web Content

CSS (Cascading Style Sheets) is the language we use to style the visual presentation of an HTML document. While HTML gives us the "skeleton" of the page, CSS provides the "skin," "clothes," and "makeup." It controls colors, fonts, spacing, and the overall layout.

Rule Structure

A CSS "rule" is composed of a **Selector** and a **Declaration Block** (one or more declarations inside curly braces).

p {
  color: blue;
  font-size: 16px;
}
  • Selector (p): Tells the browser which HTML elements to style.
  • Declaration (color: blue;): A single property-value pair.

Applying CSS to Your HTML

  • Inline: The style attribute directly on an element. (Least recommended).
  • Internal: A <style> tag in the <head> of your document.
  • External: A separate .css file linked via the <link> tag. (Most recommended).

The Concept of "Cascading"

The "Cascading" part of CSS means that if multiple rules affect the same element, the browser uses a specific set of rules to decide which style "wins." This is based on **Importance**, **Specificity**, and **Source Order**.

Comments in CSS

You can add comments to your CSS using /* comment */. These are hidden from users but are a great way to organize your code for yourself and other developers.

💡 Pro Tip: Always use an external CSS file for a real project. This keeps your HTML clean, improves performance through browser caching, and makes it easy to update your entire site from one central location.
