CSS Flexbox: Effortless One-Dimensional Layouts

Flexbox (Flexible Box Layout) is a layout model designed for laying out items in a single dimension—either as a row or a column. It makes it incredibly easy to align elements, distribute space, and handle different screen sizes without using floats or percentages.

Container Properties

  • display: flex;: Activates flexbox on the container.
  • flex-direction: Controls the direction (row, row-reverse, column, column-reverse).
  • justify-content: Aligns items along the **Main Axis** (e.g., center, space-between, flex-start).
  • align-items: Aligns items along the **Cross Axis** (e.g., center, stretch, flex-end).
  • flex-wrap: Controls whether items wrap to a new line if there's not enough room.

Flex Item Properties

  • flex-grow: Tells an item how much it should grow relative to its neighbors if there's extra space.
  • flex-shrink: Tells an item how much it should shrink if there's not enough space.
  • flex-basis: Sets the initial size of an item before growing or shrinking.
  • order: Changes the visual order of an item without changing the HTML.

The gap Property

Instead of adding margins to every child item, you can use the gap property on the flex container to set the spacing between all children automatically.

💡 Pro Tip: Flexbox is "one-dimensional". Use it when you need to align things in a line (like a navbar or a card's footer). For complex two-dimensional layouts (multiple rows and columns that align), CSS Grid is usually the better choice.
