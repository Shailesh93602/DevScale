CSS Grid: Master Two-Dimensional Layouts

CSS Grid is a powerful layout system designed for two-dimensional layouts—managing both rows and columns at the same time. It allows you to create complex, responsive website structures with very little code.

Container Properties

  • display: grid;: Activates the grid on the container.
  • grid-template-columns / grid-template-rows: Defines the size and number of columns/rows (e.g., repeat(3, 1fr)).
  • grid-template-areas: A visual way to name areas of your grid (e.g., "header header", "sidebar main").
  • gap: Sets the spacing between all grid cells.

Grid Item Properties

  • grid-column / grid-row: Tells an item which lines it should start and end on (e.g., grid-column: 1 / 3;).
  • grid-area: Assigns an item to a named area defined in grid-template-areas.

The fr (Fractional) Unit

The fr unit is unique to CSS Grid. It represents a fraction of the available space in the grid container, making it easy to create perfectly flexible columns without calculating percentages.

💡 Pro Tip: Use grid-template-areas for high-level page layouts (header, sidebar, main, footer). It makes your CSS much easier to read and understand at a glance compared to using line numbers.
