CSS Positioning: Placing Elements Precisely

While the standard document flow places elements one after another, the position property allows you to break out of that flow to place elements exactly where you need them.

The Five Positioning Values

  • Static (Default): Elements are positioned according to the normal document flow. top, left, etc., have no effect.
  • Relative: The element is positioned relative to its *original* position. It still takes up the original space in the document flow.
  • Absolute: The element is removed from the document flow and positioned relative to its *closest positioned ancestor* (anything other than static).
  • Fixed: The element is positioned relative to the browser window. It stays in the same place even when you scroll.
  • Sticky: A hybrid of relative and fixed. The element behaves like relative until it reachers a specific threshold, then "sticks" to the page like fixed.

The Importance of Z-Index

When you start positioning elements, they might overlap. The z-index property controls which element is "on top." A higher number means the element is closer to the viewer.

💡 Pro Tip: If an absolute element isn't staying where you expect, check its parent. Remember: absolute positions itself relative to the parent *if* that parent has position: relative; (or absolute/fixed) applied to it.
