CSS Shorthand Properties: Writing Less, Doing More

CSS shorthand properties allow you to set the values of multiple other CSS properties simultaneously. They keep your code clean, readable, and reduce the overall file size of your stylesheets.

1. Margin and Padding

Instead of writing four separate lines for top, right, bottom, and left, you can combine them all into one line.

margin: 10px; /* All sides 10px */
margin: 10px 20px; /* Top/Bottom 10px, Left/Right 20px */
margin: 10px 20px 30px 40px; /* Top, Right, Bottom, Left (Clockwise) */

2. Font Shorthand

The font property can bundle `size`, `weight`, `family`, and `line-height` into a single declaration.

font: bold 16px/1.5 'Inter', sans-serif;

3. Background Shorthand

This is arguably the most powerful shorthand, allowing you to set an image, color, repeat behavior, and position all at once.

background: #eee url('image.jpg') no-repeat center / cover;

4. Border and Outline

Quickly define the width, style, and color of your borders and outlines.

border: 2px solid #333;

5. Transition and Animation

Bundling `duration`, `timing-function`, and `delay` into one line makes complex animations much easier to manage.

transition: transform 0.3s ease-in-out;

💡 Pro Tip: Be careful with shorthand! It can "reset" any properties you don't explicitly mention. For example, if you use background: red;, it will automatically remove any background images you previously set on that element.
