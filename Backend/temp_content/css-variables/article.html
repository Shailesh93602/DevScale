CSS Variables (Custom Properties): Streamline Your Styles

CSS Variables (also called **Custom Properties**) allow you to store specific values (like colors, spacing, or fonts) in one central place and reuse them throughout your entire stylesheet. This makes your code more readable and incredibly easy to maintain.

Defining Variables

Variables are defined using a double-dash prefix (--). They are usually declared inside the :root selector so they are available everywhere on your page.

:root {
  --primary-color: #3498db;
  --main-padding: 1rem;
}

Using Variables (The var() Function)

To apply your variable to a CSS property, wrap it in the var() function.

.button {
  background-color: var(--primary-color);
  padding: var(--main-padding);
}

Dark Mode and Theme Switching

One of the most powerful uses for CSS variables is "Theming." If you want to create a Dark Mode, you don't have to rewrite all your CSS. You just change the values of your variables inside a specific class or media query.

[data-theme="dark"] {
  --primary-color: #2c3e50;
  --bg-color: #000;
}

Dynamic Updates with JavaScript

Unlike preprocessor variables (like SASS), CSS variables are "live." You can actually update them using JavaScript, which allows for advanced interactive effects that were previously impossible.

💡 Pro Tip: Use Fallbacks! If a variable might not be defined (e.g., if it's being loaded from an external file), you can provide a second "fallback" value inside the var() function: var(--color, blue).
