HTML Basics: The Foundation of the Web

HTML, or HyperText Markup Language, is the standard markup language used to create the structure of web pages. It is the skeletal framework that holds everything together, from text and images to links and interactive elements.

What is HTML?

Unlike a programming language that deals with logic and calculations, HTML is a markup language. It uses "tags" to annotate text, telling the web browser how to display different pieces of content. For example, you use tags to define headings, paragraphs, lists, and more.

The Anatomy of an HTML Element

Most HTML elements follow this structure:

  • Opening Tag: Tells the browser where the element starts (e.g., <p>).
  • Content: The actual text or data inside.
  • Closing Tag: Tells the browser where the element ends (e.g., </p>).

Together, these three parts form an HTML Element.

Why is Semantic HTML Important?

Using the correct tags (e.g., using <header> for the top section instead of a plain <div>) is called Semantic HTML. This is crucial for SEO (Search Engine Optimization) and Accessibility, as it helps screen readers and search engines understand the meaning of your content.

💡 Pro Tip: Always remember to close your tags! While some browsers are forgiving, unclosed tags can cause layout breaks and accessibility issues.

Tag Description

<p> Defines a paragraph of text.

<img> Embeds an image in the document (self-closing).
