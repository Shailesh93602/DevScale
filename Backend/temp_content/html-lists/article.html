HTML Lists: Grouping Related Items

Lists are used for grouping many related items together on a page. HTML supports three main types of lists: Unordered, Ordered, and Description lists, each serving a specific structural purpose.

1. Unordered Lists (bulleted)

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default.

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

2. Ordered Lists (numbered)

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers by default.

<ol>
  <li>Preheat oven</li>
  <li>Mix ingredients</li>
  <li>Bake for 30 minutes</li>
</ol>

3. Description Lists

A description list is a list of terms, with a description of each term. It is useful for glossaries or metadata. It uses three tags:

  • <dl>: Defines the description list.
  • <dt>: Defines the term (name).
  • <dd>: Describes each term.
<dl>
  <dt>HTML</dt>
  <dd>A markup language for the web.</dd>
  <dt>CSS</dt>
  <dd>A style sheet language for the web.</dd>
</dl>

Nested Lists

You can create nested lists by placing a new list (<ul> or <ol>) inside a list item (<li>) of another list.

💡 Semantic Tip: Use <ul> when the order of items doesn't matter (like a grocery list) and <ol> when the sequence is important (like a set of instructions).
