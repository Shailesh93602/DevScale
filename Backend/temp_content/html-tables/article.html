HTML Tables: Organizing Tabular Data

Tables are used to display data in a grid-like format of rows and columns. While they were historically used for page layouts, modern web design uses them strictly for tabular data (like schedules, price lists, or inventory data).

The Core Structure

A basic HTML table consists of the <table> element along with one or more <tr>, <th>, and <td> elements.

  • <table>: The parent container.
  • <tr>: Table Row.
  • <th>: Table Header (text is usually bold and centered).
  • <td>: Table Data (a standard table cell).

Example of a Simple Table

<table>
  <tr>
    <th>Subject</th>
    <th>Score</th>
  </tr>
  <tr>
    <td>HTML</td>
    <td>95</td>
  </tr>
</table>

Advanced Table Sections

For larger tables, you should organize your rows into three distinct semantic sections:

  • <thead>: Groups the header content.
  • <tbody>: Groups the main body content.
  • <tfoot>: Groups footer content (e.g., totals or summaries).

Spanning Columns and Rows

If you need a cell to cover more than one column or row, use these attributes:

  • colspan: Specifies the number of columns a cell should span.
  • rowspan: Specifies the number of rows a cell should span.

💡 Accessibility Tip: Always use <th> for header cells. This allows screen readers to associate the data in the cells with their corresponding headers, making the table much easier to navigate for visually impaired users.
