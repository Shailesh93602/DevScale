Images and Multimedia: Bringing Pages to Life

Visual content is a core part of the modern web experience. HTML provides tags not only for static images but also for rich media like audio and video, allowing for immersive and interactive web pages.

Embedding Images with <img>

The <img> tag is used to embed an image. It is an empty (self-closing) element and requires two essential attributes:

  • src: The path to the image file.
  • alt: Descriptive text for the image (essential for accessibility).
<img src="logo.png" alt="EduScale Logo" width="200" height="100">

The <figure> and <figcaption> Tags

To provide a caption for an image, it's best practice to wrap the image in a <figure> element and use <figcaption> for the text.

<figure>
    <img src="skyline.jpg" alt="New York City Skyline">
    <figcaption>The stunning skyline of New York City at sunset.</figcaption>
</figure>

Working with Video

The <video> tag allows you to embed video files directly into your page. You can include controls (play, pause, volume) using the controls attribute.

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Working with Audio

Similarly, the <audio> tag is used for sound files. It also supports the controls attribute.

<audio controls>
    <source src="podcast.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

💡 Performance Tip: Always specify width and height for images to prevent "layout shift" as the page loads. Also, use modern formats like WebP for better compression without losing quality.
