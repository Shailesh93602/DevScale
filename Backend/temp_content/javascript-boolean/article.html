JavaScript Boolean: The Logic of True and False

In programming, a **Boolean** is a simple data type that can have only two possible values: true or false. Booleans are the logical "switches" that control every decision your website makes.

1. What is a Boolean?

Used to represent "Yes-No" or "On-Off" states. For example: Is the user logged in? Is the dark mode active? Is the form data valid?

let isLoggedIn = true;
let isDarkTheme = false;

2. Comparisons (Creating Booleans)

Most booleans are created by comparing two other values using operators like >, <, ===, and !==.

let checkAge = (userAge >= 18); // returns true or false

3. The Boolean() Function

You can use the Boolean() function to check if any piece of data is "Truthy" or "Falsy."

  • Truthy Values: Almost anything with content (a number like 1, a string like "Hello", an object).
  • Falsy Values: Zero (0), an empty string (""), null, undefined, and NaN.

4. Logical Operators (&&, ||, !)

These operators allow you to combine multiple boolean values together to create complex logical rules.

  • ! (Logical NOT): Reverses a boolean value (e.g., !true becomes false).
  • && (Logical AND): Returns true only if BOTH sides are true.
  • || (Logical OR): Returns true if AT LEAST one side is true.

💡 Pro Tip: Use **Boolean shorthand**! Instead of writing if (isActive === true), you can simply write if (isActive). This is cleaner and is considered a best practice in the industry.
