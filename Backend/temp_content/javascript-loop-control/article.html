JavaScript Loop Control: Repeating Yourself Efficiently

In programming, you often need to perform the same action many times (e.g., printing every user in a list or calculating the total of every item in a cart). **Loops** allow you to write the code once and repeat it as many times as you need.

1. The for Loop

The most common loop. You tell it three things: where to start, what condition to check for, and how to change the counter each time.

for (let i = 0; i < 5; i++) {
  console.log("I am repeating!");
}

2. The while Loop

A loop that keeps running as long as a certain condition is true. It's often used when you don't know exactly how many times you'll need to repeat (e.g., "Wait until the data finishes downloading").

3. The forEach Loop (Modern)

Specifically designed for **Arrays**. It allows you to run a function for every single item in the list without worrying about indexes or counters.

users.forEach(user => {
  console.log("Hello, " + user.name);
});

4. Controlling Loops (break and continue)

  • break: Immediately stops the loop entirely, even if the condition is still true.
  • continue: Skips the *current* repetition and moves straight to the next one.

The Infinite Loop Warning

If your loop's condition is *always* true, your program will never stop repeating! This will cause your browser to "freeze" or crash. Always ensure your loop has a clear way to finish.

💡 Pro Tip: For modern web development, you should almost always use .forEach(), .map(), or for...of loops instead of the traditional C-style for loop. They are much harder to break and easier for others to read.
