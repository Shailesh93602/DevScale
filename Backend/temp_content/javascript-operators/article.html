JavaScript Operators: Calculating and Comparing Data

Operators are symbols you use in your code to perform actions on one or more pieces of data (called operands). Whether you're doing simple math or complex logic, operators are the tools that handle the work.

1. Arithmetic Operators (+, -, *, /)

These are the standard math symbols for adding, subtracting, multiplying, and dividing numbers. JavaScript also has the **Remainder (Modulus, %)** operator, which returns the amount left over after a division.

let total = 50 + 20; // 70
let remainder = 10 % 3; // 1 (10 / 3 = 3 with 1 left over)

2. Assignment Operators (=, +=, -=)

Used to assign a value to a variable. The most common is =. Shorthand operators like += are great for adding a value to a variable's current value.

let score = 10;
score += 5; // score is now 15

3. Comparison Operators (==, ===, >, <)

Used to compare two values and return true or false. **Important:** Always use the Strict Equality (===) operator, which checks if both the value AND the data type are exactly the same.

5 == "5"; // true (loose equality)
5 === "5"; // false (strict equality - RECOMMENDED!)

4. Logical Operators (&&, ||, !)

Used to combine multiple comparisons together (e.g., "IF the user is logged in AND they are an admin").

  • && (AND): Returns true only if both comparisons are true.
  • || (OR): Returns true if at least one comparison is true.
  • ! (NOT): Reverses a boolean (e.g., !true is false).

💡 Pro Tip: Always use Strict Equality (===) instead of loose equality (==). It prevents strange errors that occur when JavaScript tries to "helpfully" convert data types for you during a comparison.
