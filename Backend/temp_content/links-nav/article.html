Links and Navigation: Connecting the Web

The "Hypertext" in HTML refers to text that contains links to other text. Without links, the web would just be a collection of isolated documents. Links enable users to navigate between pages, jump to different sections on the same page, or even open a brand-new website.

The <a> (Anchor) Tag

To create a link, we use the <a> tag with the href (Hypertext Reference) attribute. This attribute tells the browser where the link should go.

<a href="https://www.google.com">Visit Google</a>

Absolute vs. Relative URLs

There are two primary ways to specify a link's destination:

  • Absolute URLs: The full web address (e.g., https://example.com/page.html). Use these for links to other websites.
  • Relative URLs: A path relative to the current folder (e.g., contact.html or /services/design.html). Use these for internal navigation within your own site.

Navigating Within a Page (Anchors)

You can create links that jump to a specific part of the current page using the id attribute of another element.

<!-- The link -->
<a href="#section-2">Go to Section 2</a>

<!-- The target -->
<h2 id="section-2">Section 2 Content</h2>

Opening in a New Tab

If you want a link to open in a new tab without closing your own page, use the target="_blank" attribute. For security reasons, you should also include rel="noopener noreferrer".

<a href="https://external.com" target="_blank" rel="noopener noreferrer">Read More</a>

💡 Accessibility Tip: Avoid using "click here" as link text. Instead, use descriptive text like "Download the PDF Guide" so that screen reader users know exactly what to expect when they click.
