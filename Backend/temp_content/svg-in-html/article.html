Using SVG in HTML: High-Performance Vector Graphics

SVG (Scalable Vector Graphics) is a language for describing two-dimensional vector graphics. Unlike standard image formats like JPEG or PNG, SVGs are written in XML code. This means they are fully searchable, scalable to any size without losing quality, and can be styled or animated using CSS and JavaScript.

Ways to Include SVG in HTML

There are three main ways to add an SVG to your web page:

1. Inline SVG

You can paste the <svg> code directly into your HTML. This is the most powerful method because it allows you to style individual parts of the graphic with CSS and interact with them using scripts.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" class="my-circle" />
</svg>

2. Using the <img> Tag

You can reference an SVG file just like any other image. This is simple and good for performance (caching), but you lose the ability to style the internal parts of the SVG with your main CSS.

<img src="icon.svg" alt="Company Logo">

3. Using a CSS Background

You can use an SVG as a background-image in your CSS. This is useful for decorative elements and patterns.

.header {
  background-image: url('pattern.svg');
}

Styling SVG with CSS

When using inline SVG, you can use specialized CSS properties:

  • fill: Sets the interior color (like background-color).
  • stroke: Sets the border color.
  • stroke-width: Sets the thickness of the border.

Why Use SVG?

  • Infinite Scalability: SVG looks crisp on everything from a watch to a 4K monitor.
  • Accessibility: SVG elements can contain <title> and <desc> tags for screen readers.
  • Small File Size: For simple icons and logos, SVG files are much smaller than their pixel-based counterparts.

💡 Pro Tip: Use an optimizer like SVGO (or the SVGO-web interface) before adding an SVG to your project. It removes unnecessary metadata and can often reduce file size by 50% or more.
