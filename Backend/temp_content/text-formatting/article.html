HTML Text Formatting: Stylizing Content Semantically

HTML provides several elements to format text on a web page. While CSS is used for most styling, certain HTML tags are used to give text specific semantic meaning (importance, emphasis, etc.) which is crucial for accessibility.

Common Formatting Tags

Tag Result Usage
<b> Bold text Used for visual bolding without extra importance.
<strong> Strong text Indicates that the text has strong importance (semantically bold).
<i> Italic text Used for alternative voice or mood (e.g., technical terms).
<em> Emphasized text Indicates emphasis (semantically italicized).

Highlighting and Marking

To draw attention to specific parts of a sentence, you can use these specialized tags:

  • <mark>: Used to highlight text, typically with a yellow background.
  • <small>: Defines smaller text, often used for side-comments or copyright notices.
  • <del>: Represents deleted text (usually with a strikethrough).
  • <ins>: Represents inserted text (usually underlined).

Subscripts and Superscripts

These are useful for mathematical or chemical formulas:

  • Subscript (<sub>): Used for H2O.
  • Superscript (<sup>): Used for E = mc2.

💡 Note: Modern web design favors using <strong> and <em> over <b> and <i> because they provide better information to screen readers used by visually impaired users.
