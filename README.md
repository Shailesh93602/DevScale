# EduScale

## All-in-One Engineering Learning Platform

EduScale is a comprehensive EdTech platform for engineering students, featuring personalized career roadmaps, coding challenges, battle zones, community forums, placement preparation, and interactive learning experiences.

### Tech Stack

**Frontend**
- Next.js 15, React 19, TypeScript
- Tailwind CSS, Radix UI, Framer Motion
- Redux Toolkit, Zustand
- Supabase Auth, Socket.io

**Backend**
- Express.js, Node.js, TypeScript
- Prisma ORM, PostgreSQL
- Socket.io, Redis, Bull Queues
- Swagger API Documentation

### Key Features

- Personalized career roadmaps with progress tracking
- Coding challenges with multi-language support
- Real-time coding battle zone with leaderboards
- Community forums and discussion threads
- Placement preparation with interview questions
- Resource library with articles and tutorials
- Achievement system and streak tracking
- Dark/light theme support

### Getting Started

```bash
# Frontend
cd Frontend
npm install
npm run dev

# Backend
cd Backend
npm install
npm run dev
```
